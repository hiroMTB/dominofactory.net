dial, 2013 update pt1, Hiroshi Matoba

Each circle/polygon shape sends MIDI message when indicator pass through its vertex point.

Red shape: note on message
Yellow shape: velocity (2nd from top left)
Light yellow shape: duration (3rd from top left)
Orange shape: note number
Green shape: volume

Channel Ring(big circle with numbers) is for MIDI output ch settings
Value Ring(small circle with numbers) is for minimum MIDI value setting
Each shape synthesize rhythm, melody and harmony in different rotation period.
